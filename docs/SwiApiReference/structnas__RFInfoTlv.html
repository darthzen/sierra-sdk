Detailed Description

This structure contains paramaters RF Band Information List.

Parameters
TlvPresentindicating the presence of the TLV in the QMI ind
radioInterfaceSizeNumber of sets of the following elements
  • radioInterface
  • activeBandClass
  • activeChannel
radioInterfaceradio interface technology of the signal being measured
  • 0x01 - cdma2000 1X
  • 0x02 - cdma2000 HRPD (1xEV-DO)
  • 0x03 - AMPS
  • 0x04 - GSM
  • 0x05 - UMTS
  • 0x08 - LTE
  • 0x09 - TD-SCDMA
activeBandClassactive band class
  • 00 to 39 - CDMA band classes
  • 40 to 79 - GSM band classes
  • 80 to 91 - WCDMA band classes
  • 120 to 160 - LTE band classes
  • 200 to 205 - TD-SCDMA band classes
activeChannelactive channel
  • Active channel. If the channel is not relevant to the technology, a value of 0 is returned.
