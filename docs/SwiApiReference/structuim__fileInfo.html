Detailed Description

This structure contains paramaters for file Information

Parameters
fileID
  • This is Identifier to SIM files; e.g. in UIM "6F07" is Identifier of IMSI File
pathLen
  • Length of file Path
path
  • Path value. This value must be the complete path of the file, which is a sequence block of 2 bytes (e.g., 0x3F00 0x7FFF).
