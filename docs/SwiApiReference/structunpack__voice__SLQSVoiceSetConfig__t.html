Detailed Description

This structure contains information about the Set Configuration Response Parameters.

Parameters
pAutoAnsStatus
  • Auto Answer Status. (optional)
  • Values:
    • 0x00 - Information was written successfully
    • 0x01 - Information write failed
    • 0xFF - Not Available.
    • NULL pointer - Invalid data.
