Detailed Description

This structure contains the TDSCDMA Signal Strength Info Extended

Parameters
rssi
  • Measured RSSI in dB
rscp[Optional]
  • Measured RSCP in dBm
ecio[Optional]
  • Measured ECIO in dBm.
sinr[Optional]
  • Measured SINR in dB. -15 dB is sent to clients if the actual SINR is less than -15 dB
