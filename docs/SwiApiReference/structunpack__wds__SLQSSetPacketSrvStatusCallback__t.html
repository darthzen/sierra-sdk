Detailed Description

This structure contains unpack set packet service status callback information.

Parameters
conn_statusconnection status.
  • Current link status. Values:
    • 1 - DISCONNECTED
    • 2 - CONNECTED
    • 3 - SUSPENDED
    • 4 - AUTHENTICATING
    • 0xff Invalid data.
reconfigReqdIndicates whether the network interface on the host needs to be reconfigured.
  • Values:
    • 0 - No need to reconfigure.
    • 1 - Reconfiguration required.
    • 0xff - Invalid data.
sessionEndReasonCall End Reason
  • See qaGobiApiTableCallEndReasons.h for Call End Reason
  • 0xffff - invalid data.
verboseSessnEndReasonTypeVerbose call end reason type
  • Call end reason type. Values:
    • 0 - Unspecified
    • 1 - Mobile IP
    • 2 - Internal
    • 3 - Call Manager defined
    • 6 - 3GPP Specification defined
    • 7 - PPP
    • 8 - EHRPD
    • 9 - IPv6
    • 0xffff - invalid data.
verboseSessnEndReasonReason the call ended (verbose)
  • See qaGobiApiTableCallEndReasons.h for Call End Reason
  • 0xffff - invalid data.
ipFamilyIP family of the packet data connection.
  • Values
    • 4 - IPv4
    • 6 - IPv6
    • 0xff - Invalid data.
techNameTechnology name of the packet data connection.
  • Values
    • 32767 - CDMA
    • 32764 - UMTS
    • 30592 - EPC
    • 30590 - EMBMS
    • 30584 - Modem Link Local EPC is a logical interface to support LTE/eHRPD handoff. It is returned if the device supports IP session continuity. Modem Link Local is an interface for transferring data between entities on the AP and modem.
    • 0xffff - Invalid data.
bearerID
  • bearer ID (3GPP) or RLP ID (3GPP2) of the packet data connection.
  • Valid Values - 0 to 16
  • 0xFF - Invalid value.
