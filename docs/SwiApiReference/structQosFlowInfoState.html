Detailed Description

This structure contains QoS flow state

Parameters
idQoS identifier
isNewFlow
  • 1 – Newly added flow
  • 0 – Existing flow
stateThis indicates that the flow that was added/modified/deleted:
  • 0x01 – Flow activated
  • 0x02 – Flow modified
  • 0x03 – Flow deleted
  • 0x04 – Flow suspended
  • 0x05 – Flow enabled
  • 0x06 – Flow disabled
