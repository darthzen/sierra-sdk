Detailed Description

This structure contains the parameters for Physical Carrier aggregation of Pcell Information.

Parameters
pci
  • Physical cell ID of the SCell Range.
  • Range for ID values: 0 to 503.
freq
  • Frequency of the absolute cell Range.
  • Range for ID values: 0 to 65535.
dl_bw_value
  • Downlink Bandwidth Values.
  • See NAS_LTE_CPHY_CA_BW_NRB_LITE for more information.
scell_state
  • Scell state Values.
  • See NAS_LTE_CPHY_SCELL_STATE_LITE for more information.
TlvPresent
  • Tlv Present.
