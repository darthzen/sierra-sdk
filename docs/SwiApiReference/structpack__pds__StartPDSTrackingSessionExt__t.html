Detailed Description

Structure contain the parameter for PDS tracking session.

Parameters
sessionControl
  • Control method:
    • 0x0 - Manual
sessionType
  • Type:
    • 0x0 - New
sessionOperation
  • Operating mode:
    • 0x00 - Standalone
    • 0x01 - MS-based
sessionServerOption
  • Location server option:
    • 0x0 - Default
fixTimeout
  • Maximum time to work on each fix (in seconds, max 255)
fixCount
  • Count of position fix requests for this session (must be at least 1)
fixInterval
  • interval between position fix requests (in seconds)
fixAccuracy
  • Preferred accuracy threshold(in meters)
