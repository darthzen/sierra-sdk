Detailed Description

This structure contains the SLQSGetAudioPathConfig response parameters.

Parameters
pECMode[Optional]
  • AV_EC
    • 0 - Echo cancellation off
    • 1 - Handset mode
    • 2 - Headset mode
    • 3 - Car kit mode
    • 4 - Speaker Mode
pNSEnable[Optional]
  • AV_NS
    • 0 - Noise suppression off
    • 1 - Noise suppression on
