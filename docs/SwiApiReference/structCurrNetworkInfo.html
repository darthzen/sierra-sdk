Detailed Description

Network information structure

Parameters
NetworkType
  • Values:
    • 0 - 3GPP
    • 1 - 3GPP2
RATMask
  • Radio Access Technology (RAT) mask to indicate the type of technology.
  • Values:
    • 0 - Don't Care
    • 0x8000 - NULL Bearer
  • CDMA RAT mask values:
    • 0x01 - CDMA_1x
    • 0x02 - EVDO_REV0
    • 0x04 - EVDO_REVA
    • 0x08 - EVDO_REVB
    • 0x10 - EHRPD
    • 0x20 - FMC
  • UMTS RAT mask values:
    • 0x01 - WCDMA
    • 0x02 - GPRS
    • 0x04 - HSDPA
    • 0x08 - HSUPA
    • 0x10 - EDGE
    • 0x20 - LTE
    • 0x40 - HSDPA+
    • 0x80 - DC_HSDPA+
    • 0x100 - 64_QAM
    • 0x200 - TDSCDMA
SOMask
  • Service Option (SO) mask to indicate the service option or type of application.
  • Values:
    • 0 - Don't Care
  • CDMA 1x SO mask values:
    • 0x01 - CDMA_1X_IS95
    • 0x02 - CDMA_1X_IS2000
    • 0x04 - CDMA_1X_IS2000_REL_A
  • CDMA EV-DO Rev 0 SO mask values:
    • 0x01 - DPA
  • CDMA EV-DO Rev A SO mask values:
    • 0x01 - DPA
    • 0x02 - MFPA
    • 0x04 - EMPA
    • 0x08 - EMPA_EHRPD
  • CDMA EV-DO Rev B SO mask values:
    • 0x01 - DPA
    • 0x02 - MFPA
    • 0x04 - EMPA
    • 0x08 - EMPA_EHRPD
    • 0x10 - MMPA
    • 0x20 - MMPA_EHRPD
